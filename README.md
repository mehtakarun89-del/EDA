# 🛒 BigBasket Product Data Analysis

✨ A simple **Data Analysis project** using **Python, Pandas, Seaborn, and Matplotlib** to explore and visualize product data from the BigBasket dataset.

This project demonstrates how to:
- Load datasets 📂
- Clean and inspect data 🔍
- Detect missing values ⚠️
- Perform exploratory data analysis 📊
- Create beautiful visualizations 📈

---

# 🚀 Technologies Used

| Tool | Purpose |
|-----|------|
| 🐍 Python | Programming Language |
| 🐼 Pandas | Data Loading & Manipulation |
| 📊 Seaborn | Statistical Data Visualization |
| 📉 Matplotlib | Advanced Plot Customization |
| 📓 Google Colab / Jupyter Notebook | Running the Analysis |

---

# 📂 Dataset

The dataset used contains **BigBasket product information** such as:

- Product Name
- Category
- Sub Category
- Brand
- Sale Price
- Market Price
- Product Type
- Rating
- Description
